Before writing any code, we will need to first set up Haxe on our computer. 
## The Compiler
A compiler is a piece of software that converts computer code from one language to another. Some languages like C have multiple different options, but for Haxe there is just the one. Lets install it
### Installing the compiler
1. First, go to the [Haxe website](https://haxe.org/). It should roughly look like the screenshot below. If it looks different, don't worry, the instructions should carry over.
   ![[Haxe Website Screenshot.png]]
2. Hit the button to download the latest stable release, in the screenshot above it is ``4.3.7``, however it may have a newer release depending on when you do this. You may also see nightly or preview versions, these contain the latest and greatest features of Haxe, but may have bugs or incomplete features not present in the stable release.
3. You should then see a list of different methods of install. The one you use will depend on your operating system, on Windows or macOS you will likely want to use the installer instead of downloading binaries. Download it and go through the installation process.
	1. Linux users can usually install from their package manager. On Fedora for instance you can simply do ``sudo dnf install haxe``. Ubuntu will require installing a PPA.
   ![[Haxe Platform Downloads.png]]

You should now have the Haxe compiler installed!
## The Editor
Haxe, like most programming languages, can be written in any text editor you choose. Notepad or TextEdit would both work just fine, however text editors specific for programming tend to be the most ideal. They add features like syntax highlighting, which colors specific parts of your code to make it more readable, and autocomplete. Programmers will argue about [which specific text editor for decades](https://en.wikipedia.org/wiki/Editor_war), but the most popular currently is Microsoft's VSCode. It also tends to have the best support for Haxe features, so we will use that.
### Installing the editor
The installation for VSCode is going to be quite similar to installing Haxe so I will leave you on your own to install it. Just make sure you install it from the official site, the URL is https://code.visualstudio.com/. It should look somewhat like the provided screenshot once you open it.
![[VSCode Getting Started.png]]

### Installing the Haxe extension
If you put your attention to the left side of VSCode, you will see the *Activity Bar* containing a bunch of icons laid out vertically. If you hover over them, you will see a tooltip showing the title of the specific activity. Press the extensions icon with the four squares. A sidebar will pop up with a list of different extensions, search up "Haxe" and install the first one:
![[VSCode Extension.png]]
If you so wish, you can grab other extensions here too. If there is a specific color scheme you want, that would be considered an extension too.
